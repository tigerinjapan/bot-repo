# ✍️ CSS プロパティの並び順のルール

## 目的

### コードの高速な理解

- プロパティが規則的に並んでいる
- 要素の**レイアウト（配置）**、**サイズ**、**見た目**が瞬時に把握できる

### 開発者間の統一

- 複数の人が関わるプロジェクトで、誰もが同じルールで記述する
- コードレビューや引継ぎがスムーズになる

### プロパティの見落とし防止

- 必要なプロパティがどこに書かれているかが明確になる
- 修正や追加の際に見落としを防ぎ、バグを減らす

## メリット

### 検索性の向上

- 特定のプロパティを探す際、配置のルールが分かっていれば迷うことなく見つけられる

### 差分（Diff）の最小化

- バージョン管理システムで変更履歴で、修正した箇所だけが差分として表示される
- プロパティの順番が変わらないため、管理しやすくなる

### メンテナンス性の向上

- コードが整理され、誰が読んでも理解しやすくなる
- 修正や機能追加のコストが下がる

## 並び順

| カテゴリ           | プロパティ                 | 説明                         | 備考                    |
| :----------------- | :------------------------- | :--------------------------- | :---------------------- |
| **表示・配置**     | `display`                  | 要素の表示形式を決定         | 最も重要。最初に配置    |
|                    | `position`                 | 配置方法（固定など）を決める | `top`などとセットで使用 |
|                    | `top, right, bottom, left` | 配置の具体的な位置           |                         |
|                    | `z-index`                  | 要素の重なり順序             | 値が大きいほど手前      |
| ---                | ---                        | ---                          | ---                     |
| **ボックスモデル** | `width, height`            | 要素の**幅と高さ**           | サイズを決める          |
|                    | `margin`                   | 要素の**外側の余白**         | 外側の間隔調整          |
|                    | `padding`                  | 要素の**内側の余白**         | 内側の間隔調整          |
|                    | `overflow`                 | サイズ超過時の処理           | はみ出た部分の制御      |
| ---                | ---                        | ---                          | ---                     |
| **Flex/Grid**      | `gap`                      | Flex/Grid 要素間の間隔       |                         |
|                    | `flex`                     | Flexbox の動作を制御         |                         |
|                    | `grid`                     | Grid レイアウトの動作を制御  |                         |
| ---                | ---                        | ---                          | ---                     |
| **文字・フォント** | `font`                     | フォントの種類やサイズ       |                         |
|                    | `text`                     | 文字の揃えや装飾             |                         |
|                    | `line-height`              | 行の高さ（行間）             |                         |
|                    | `white-space`              | **スペースや改行の扱い**     |                         |
| ---                | ---                        | ---                          | ---                     |
| **背景・色**       | `background-color`         | 要素の**背景色**             | `background`でも OK     |
|                    | `color`                    | テキストの**文字色**         |                         |
|                    | `border`                   | 要素の**境界線（枠）**       |                         |
|                    | `box-shadow`               | 要素に影をつける             |                         |
| ---                | ---                        | ---                          | ---                     |
| **その他**         | `transition`               | 変化を滑らかにする           | アニメーション効果      |
|                    | `animation`                | 複雑な動きを設定             |                         |
|                    | `cursor`                   | マウスカーソルの形           |                         |
|                    | `opacity`                  | 要素の**透明度**             |                         |
