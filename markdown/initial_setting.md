# pip コマンド

## ライブラリ・インストール

```
pip install -r requirements.txt
```

## ライブラリ・アップグレード

```
pip install -U pip
pip-review --auto
```

## インストール・ライブラリ・ファイル出力

```
pip list -o > "requirements.txt"
```

## 拡張機能確認

```
code --list-extensions --show-versions
```
